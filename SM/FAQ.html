How to add Scrims Manager on my server?

You can invite him by clicking on the "INVITE" in the top right corner of the page or invite him directly.

How many lobbies i can make with Scrims Manager?

You can make up to 4 lobbies. If you need more lobbies, contact @HappyJuice on the support server.

How can i do results with bot?

1) activate results -> b ractive

2) get link to spreadsheets -> b link

3) update teams in spreadsheet according to current slot list -> b sheet

4) insert placement and kills of each team in each match in spreadsheet

5) once all data are in, use b results a to get results of lobby A -> b results a

I want to add my own graphic to results. Where i can do that?

1) Write b config and click on the "General" category

2) Go to the "Results" section

3) After bot reacts/sends a message with some info, just upload your graphic/image.

Scrims Manager doesn't work on my server. What should i do?

1) Look at Scrims Manager status. If you see "loading services, please wait" in his status wait 10-15 minutes and Scrims Manager will work again

2) Check if Scrims Manager has access to the channel

3) Check permissions of Scrims Manager role (if Scrims Manager has more roles, check all the roles he has)

4) If you did all of this (or problem continues) please join support server and ask for help there

When teams have same points, why team with more kills is lower and team with more place points is higher?

Because calculation of points is based on official competition rules (page 30, article 2.2) placement points have priority.

Where i can get source code of Scrims Manager?
