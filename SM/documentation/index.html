Getting started

Starting guide of Scrims Manager

We recommend having at least these roles, channels and messages prepared before starting configuration (Because you need to have these roles and channels for Scrims Manager working):

  • Channels:
  • Lobby
  • Registration
  • Announcement
  • Accept teams [ADMIN channel]

  • Roles:
  • Lobby
  • Scrims access
  • Announcement

  • Messages:
  • Notification
  • Scrims access Registration open
  • Slots confirmation
