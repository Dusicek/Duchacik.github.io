Emojis

Explanation of the Emojis settings category. This category is fully optional = you don't need to configure anything.

Accept / Confirm

Configuration of emoji, that will be used for confirming and accepting.

Reject / Cancel

Configuration of emoji, that will be used for cancelling and rejecting.

Warning

Configuration of emoji that will be used for warning you, when something is wrong (wrong format of registration or roster, user is banned in scrims, wrong data when configurating bot).

Slots

Configuration of emojis/numbers in Slots message. You need to react with emoji you want to bot message. Pay attention to the bot message, because you can see what slot you are configurating!!!

Reset all emojis

All emojis will be set to default.
